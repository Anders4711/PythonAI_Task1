# Hangman Game (GUI)

A simple Hangman game with a graphical user interface.

## Features
- Random word selection from a predefined list  
- Graphical display of the word as underscores  
- Visual indication of correct and incorrect guesses  
- Limited number of attempts before game over  
- “New Game” button to restart

## Technologies
- Language: Python  
- GUI: Tkinter (standard library)

## How to Run
1. Make sure you have Python 3 installed.
2. Clone/download the project.
3. From a terminal in the project folder, run:
python3 MainHangman.py
